# PulseMarket AI Landing Page

This is a static landing page for PulseMarket AI.

Open locally:

```bash
open landing/index.html
```

No build step, JavaScript, analytics script, token, wallet integration, or external dependency is required.

Production options:

- Serve it as a static site with Nginx on the VPS.
- Publish it through GitHub Pages.
- Attach a domain later when the Builders Program submission needs a public website.
