# Telegram Comments Setup

Telegram comments under channel posts require a linked discussion group.

Current public setup:

- Channel: [@PulseMarketAI](https://t.me/PulseMarketAI)
- Discussion / support group: [@PulseMarketAISupport](https://t.me/PulseMarketAISupport)

## Link the discussion group

1. Open `@PulseMarketAI`.
2. Open channel settings.
3. Choose **Edit**.
4. Choose **Discussion**.
5. Link the `PulseMarket AI Support` group.
6. Save.

After this, Telegram can show a comment thread under channel posts.

## Translation strategy

Main channel posts should be English-first.

Do not mix English and Russian in one public channel post. This makes Telegram translation cleaner and helps international users understand the update.

Russian users can enable user-side translation:

```text
Settings → Language → Show Translate Button
```

Telegram channel auto-translate requires channel boost level 3, but user-side translation does not.
