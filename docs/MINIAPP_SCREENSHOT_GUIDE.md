# Mini App Screenshot Guide

Use this guide for clean Builders Program screenshots of the PulseMarket AI
Telegram Mini App.

## Required Screenshots

1. Home dashboard
2. Today's Pulse hero
3. Smart Money Radar
4. Hot Markets
5. Search
6. Watchlist empty state
7. Mobile scroll view

## Capture Rules

- Capture only the Mini App UI.
- Use dark mode.
- Do not include browser chrome if possible.
- Do not include terminal windows.
- Do not include desktop files or notifications.
- Do not include private chats.
- Do not include `.env`, tokens, server logs, or personal data.

## Recommended Flow

1. Open `http://2.26.80.27:8080/app` for browser preview.
2. For Telegram review, use the Mini App only after HTTPS is configured.
3. Take vertical mobile screenshots first.
4. Keep the safety footer visible in at least one screenshot.

## Notes

PulseMarket AI is positioned as market intelligence, not trade execution. The
screenshots should emphasize discovery, Today’s Pulse, Smart Money Radar, search,
and safety scope.
